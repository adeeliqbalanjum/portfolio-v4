# Adeel Portfolio V6 — Reference Hero

This repository contains the Next.js portfolio homepage for Muhammad Adeel Iqbal.

The homepage is designed to closely match the supplied reference style:

- Minimal white canvas
- Small yellow availability badge
- Compact centered Satoshi-style headline
- Black primary CTA and quiet secondary CTA
- Center portrait card with thick rounded black border
- Layered website/project cards behind the portrait
- Soft edge blur/fade on the side cards
- Orange, purple, and green moving gradient accents
- Responsive stats, projects, and contact sections

## Portfolio positioning

Muhammad Adeel Iqbal is positioned as a WordPress Performance & WooCommerce Specialist focused on speed, conversions, bug fixing, custom development, and AI-powered workflow ideas.

## Run locally

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```
